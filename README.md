**Project Deliverables**

This project consists of the following four main deliverables:

1: Software Requirements Specification (SRS)

2: Design Document

3: Prototype

4: Final Deliverable

**Software Requirements Specification (SRS)**

The SRS document includes:

1.1: Project scope

1.2: Functional and non-functional requirements

1.3: Use case diagrams

1.4: Usage scenarios

1.5: Adopted development methodology

1.6: Work plan and project timeline

**Design Document**

The Design Document describes the system architecture and detailed design.
It is currently under development and will be updated as the project progresses.
